# Anime-World

An Anime store to search for your favourite anime with good functional features and decent-looking UI.

This app provides source links to many anime shows (even if they are not so popular).

It makes searching for shows much easier and also makes it easy to keep track of what you are wanting to watch, what is your favourite with features like watchList, favourites & much more.

It's made with Kotlin and Android-SDK & uses [Anime-Backend](https://github.com/Srikanth-Macha/Anime-Backend) as its backend(NodeJS with ExpressJS application) at the server-side ...
